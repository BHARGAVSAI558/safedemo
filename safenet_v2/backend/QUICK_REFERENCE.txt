START BACKEND
──────────────

python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000


VERIFY HEALTH
──────────────

curl http://localhost:8000/health
curl http://localhost:8000/api/v1/demo/status


CONNECT WEBSOCKET
──────────────────

Worker:
  wscat -c "ws://localhost:8000/ws/claims/1?token=<jwt>"

Admin:
  wscat -c "ws://localhost:8000/ws/admin/feed?token=<admin_jwt>"


TRIGGER DISRUPTION
────────────────────

curl -X POST http://localhost:8000/api/v1/simulation/trigger \
  -H "Authorization: Bearer <admin_jwt>" \
  -H "Content-Type: application/json" \
  -d '{"zone_id": "default", "disruption_type": "rain"}'
