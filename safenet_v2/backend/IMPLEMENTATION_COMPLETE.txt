═══════════════════════════════════════════════════════════════════════════════
  INTEGRATION FLOWS (9 complete end-to-end flows)
═══════════════════════════════════════════════════════════════════════════════

FLOW 1: Worker Onboarding → Policy Activation
  Register → Create Policy → Pay Premium → Policy Active → Eligible for Claims

FLOW 2: Scheduler → Disruption → Claims → Payout → WebSocket
  disruption_scan() → check_disruptions_for_zone() → initiate_claims_for_disruption()
  → PayoutEngine → FraudEngine → Decision → PayoutRecord → publish_claim_update()
  → Worker receives notification on /ws/claims/{worker_id}

FLOW 3: Admin Creates Social Event (Curfew/Strike)
  Admin creates DisruptionEvent → disruption_scan detects it
  → Triggers claims pipeline → Workers receive payouts

FLOW 4: Fraud Detection → Flagged Claims → Admin Visibility
  check_fraud() → FraudFlag + FraudSignal rows → Admin sees in /ws/admin/feed

FLOW 5: Stale Claim Resolver (every 5 min)
  Auto-approves claims stuck > 2 hours → Triggers payout → Updates trust score

FLOW 6: Premium Recalculation (every Monday 00:00 IST)
  Recalculates premiums via XGBoost → Updates Policy → Sends notifications

FLOW 7: Pool Health Audit (every Sunday 23:59 IST)
  Aggregates weekly payouts → Updates pool balance → Triggers reinsurance alerts

FLOW 8: Trust Score Update (daily 02:00 IST)
  Rewards clean workers (+0.02) → Penalizes inactive (-0.01) → Bounds 0.0–1.0

FLOW 9: Forecast Shield (every 6 hours + startup)
  Fetches 48h forecast → Stores in app.state → Auto-upgrades coverage before rain
